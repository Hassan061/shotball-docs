## About

Shotball is a fast-paced, team-based game where players compete in arena-style matches. This documentation covers everything from game design concepts to technical implementation details.

### Tech Stack
- **Engine:** Unreal Engine 5.4.4
- **Language:** C++ / Blueprints
- **Networking:** EOS P2P + Steam (Listen Server)
- **Backend:** AWS (Lambda, DynamoDB, Cognito) + Firebase
- **Platform:** Windows (Steam)
